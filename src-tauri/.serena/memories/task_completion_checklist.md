# タスク完了時のチェックリスト

## 必須実行コマンド

### TypeScript関連
```bash
pnpm build        # TypeScriptコンパイル + Viteビルド
```

### Rust関連
```bash
cd src-tauri
cargo test        # Rustテストの実行
cargo clippy      # linting（利用可能な場合）
```

### アプリケーション動作確認
```bash
pnpm tauri dev    # アプリケーションの動作確認
```
